# React Native CLI NativeWind Boilerplate

A simple React Native CLI boilerplate with NativeWind for styling and React Navigation for navigation.

## Features
- React Native CLI
- NativeWind (Tailwind CSS for React Native)
- React Navigation preconfigured
- Clean and minimal setup

## Getting Started
Clone the repository, install dependencies, and run the app.

```bash
git clone https://github.com/zumrakb/reactnativecli-nativewind-boilerplate.git
cd reactnativecli-nativewind-boilerplate
npm install

# Android
npx react-native run-android

# iOS
npx react-native run-ios
```
### Styling

NativeWind is configured, so you can use Tailwind-style utility classes using className in components.

### Navigation

React Navigation is already set up and ready to use.

### Purpose

Use this boilerplate to quickly start React Native CLI projects without repeating setup.
